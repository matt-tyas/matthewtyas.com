Take control of your admin dashboard. Offers customized dashboard widget visibility control for each user level.

== Description ==

Dashboard Heaven lets you get rid of those annoying feeds and widgets on your user's dashboard.

Dashboard Heaven offers
* Widget to User-level/role customisation 
* Control which user levels see which widget.
* Hide all or only a few dashboard widgets

== Installation ==

1. Upload `dashboard-heaven` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. In the Admin panel, go to Tools => Dashboard Heaven
1. Select the minimum visibility for each widget
